# 合同文件上传功能完成！

## 新增功能

### 后端
✅ **数据库更新**
- 新增 `ContractFile` 表
- 合同与文件一对多关系
- 支持级联删除（删除合同时自动删除文件）

✅ **API接口**
- `POST /api/contracts/:id/files` - 上传多个文件
- `GET /api/contracts/:id/files` - 获取合同文件列表
- `DELETE /api/contracts/:id/files/:fileId` - 删除文件
- `GET /api/contracts/files/:fileId/download` - 下载文件

✅ **文件存储**
- 使用 multer 处理文件上传
- 文件存储在 `backend/uploads/` 目录
- 支持 PDF、Word、Excel、图片等格式
- 单文件最大 10MB
- 最多可同时上传 10 个文件

### 前端
✅ **UI界面**
- 合同列表新增"文件"按钮
- 文件管理弹窗
  - 显示已上传文件列表
  - 显示文件大小和上传时间
  - 支持下载文件
  - 支持删除文件
  - 支持多文件上传

✅ **文件管理功能**
- 点击"文件"按钮打开文件管理弹窗
- 支持拖拽或点击上传多个文件
- 实时显示上传进度
- 文件列表自动刷新

## 如何启动

### 方法1：使用启动脚本（推荐）
```bash
# 在 crm-system 目录下双击运行
restart.bat
```

### 方法2：手动启动
```bash
# 1. 停止现有Node进程
taskkill /F /IM node.exe

# 2. 生成Prisma Client
cd backend
npx prisma generate

# 3. 启动后端
npm run dev

# 4. 在另一个终端启动前端
cd ../frontend
npm run dev
```

## 使用步骤

1. 启动服务后，访问 http://localhost:3000
2. 进入"合同管理"页面
3. 在合同列表中找到要管理的合同
4. 点击"文件"按钮
5. 在弹出的文件管理窗口中：
   - 点击"上传文件"选择文件（可多选）
   - 查看已上传的文件列表
   - 点击"下载"下载文件
   - 点击"删除"删除文件

## 技术细节

### 支持的文件类型
- PDF (.pdf)
- Word (.doc, .docx)
- Excel (.xls, .xlsx)
- 图片 (.jpg, .jpeg, .png, .gif)
- 文本文件 (.txt)

### 文件限制
- 单文件最大：10MB
- 单次最多上传：10个文件
- 文件名自动重命名（防止冲突）

### 安全特性
- 文件类型验证
- 文件大小限制
- 认证授权检查
- 级联删除（删除合同时自动清理文件）
